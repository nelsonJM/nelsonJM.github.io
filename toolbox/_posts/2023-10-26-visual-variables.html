Visual variables is the idea that an element, like a button, has 7 dimensions that can be "tweaked"...

  • Size
  • Shape
  • Position
  • Texture
  • Orientation
  • Value
  • Hue

We can use the button below to illustrate.

Not all of the variables have to be adjusted. Below: size, shape, position (scrolling text in 'Marquee Sign' example), value, and hue are changed to create 3 variations.
